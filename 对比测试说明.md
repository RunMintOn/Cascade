# Obsidian Canvas 对比测试

## 测试目标
找出为什么WebCanvas导出的文件无法编辑，而Obsidian手动创建的可以。

## 测试步骤

### 1. 测试从Obsidian复制JSON
1. 在Obsidian中创建一个**手动Canvas**（不要导入）
2. 添加1-2个文本节点
3. 拖动节点，确认可以编辑
4. 按 `Ctrl+S` 保存
5. 用文本编辑器打开这个Canvas文件
6. 复制整个JSON内容
7. 粘贴到这里，标记为「Obsidian手动创建的JSON」

### 2. 测试WebCanvas导出的JSON
1. 从你的日志中，找到这段：
```
[WebCanvas DEBUG] Generated JSON: {
  "nodes": [...]
}
```
2. 复制整个JSON（从 `{` 开始到 `}` 结束）
3. 粘贴到这里，标记为「WebCanvas导出的JSON」

## 我需要查看的差异
1. 字段名是否一致（id, type, x, y, width, height, text）
2. 字段顺序
3. 引号类型（单引号 vs 双引号）
4. 缩进和空格
5. 是否有隐藏字符
6. 是否有Obsidian特定的元数据
